# TaskFlow

TaskFlow is a desktop task management application built using C# Windows Forms and SQLite database.

## Features

* Add tasks
* Edit tasks
* Delete tasks
* Task priority system (Low / Medium / High)
* Task status management
* Personal tasks view
* Monthly calendar view
* Calendar navigation
* Dashboard statistics
* Task color visualization
* Inactive calendar days
* Tooltips with task descriptions
* SQLite database integration
* User login system

## Technologies

* C#
* Windows Forms
* SQLite
* .NET
* GitHub

## Application Views

### Dashboard

Displays all tasks in a styled table with priority colors.

### My Tasks

Displays only tasks assigned to the logged-in user.

### Calendar

Monthly task calendar with:

* weekly alignment
* task colors by priority
* inactive days from previous month
* task status display
* tooltips with task descriptions

### Statistics

Displays:

* total tasks
* personal tasks
* high priority tasks
* completed tasks

## Task Features

Each task includes:

* title
* description
* start date
* end date
* priority
* status
* assigned user
