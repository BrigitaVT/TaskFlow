# TaskFlow

TaskFlow is a desktop task management application built using C# Windows Forms and SQLite database.

## Features

- Add tasks
- Edit tasks
- Delete tasks
- Calendar view
- Monthly calendar navigation
- Dashboard overview
- SQLite database integration
- Task scheduling
- Task management interface

## Technologies

- C#
- Windows Forms
- SQLite
- GitHub

## Project Structure

TaskFlow/
├── Models/
├── Repositories/
├── Controls/
├── Helpers/
├── Forms/

## Application Views

### Dashboard
Displays calendar with scheduled tasks.

### Tasks
Allows users to:
- add tasks
- edit tasks
- delete tasks

### Calendar
Monthly task calendar with navigation buttons.
